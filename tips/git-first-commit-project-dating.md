# 项目时间溯源用 git 首次提交，不用文件时间戳
type: method
date: 2026-06-20
source: 简历项目经历盘点——6个项目的开始时间溯源

## 现象
需要盘点项目经历中的"项目开始时间"，直接看目录下文件的创建/修改时间，发现不准——clone 时间、拷贝时间、编辑时间都会覆盖真实起点。

## 根因
文件时间戳反映的是"这个副本什么时候出现在这里"，不是"这个项目什么时候开始的"。git 首次提交才是项目从零到一的真实时刻。

## 步骤
```bash
git log --reverse --format="%ai" | head -1
```
- `--reverse` 最早提交在前
- `--format="%ai"` 输出 ISO 8601 格式的 author date
- 非 git 仓库回退到目录最早文件时间戳，但标注方法差异

## 预防
任何需要"项目是什么时候开始的"的场景（简历、复盘、项目清单）——先跑 git log，不靠记忆也不靠文件时间戳。
